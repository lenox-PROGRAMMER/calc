# Finance Tracker

A simple finance tracker application for managing and tracking income and expenses.

## Project Overview

This project provides a basic finance tracker functionality with features to add income and expenses, view entries, and see a summary balance. It is designed as a backend Python module initially but can be extended into a full-stack web application.

## Features

- Add income entries with amounts and descriptions
- Add expense entries with amounts and descriptions
- View all incomes and expenses
- Calculate and display the balance (income - expenses)

## Tech Stack

- Python 3.x
- (Planned) Node.js with Express for backend API
- (Planned) React for frontend UI
- (Planned) PostgreSQL for database
